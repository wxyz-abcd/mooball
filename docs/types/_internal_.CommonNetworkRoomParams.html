    Properties

    Optionalconfig

    config?: RoomConfig

    The RoomConfig object that contains all the main callbacks of this room. Default value is null. Note that the api will generate an empty config if you do not provide one. Look at https://github.com/wxyz-abcd/mooball/tree/main/examples/roomConfigs for example RoomConfigs to use here.

    OptionaldebugDesync

    debugDesync?:
        | true
        | ((hostRoomState: RoomState, clientRoomState: RoomState) => void)

    For a host room, can be true. For a client room, can be a callback function with parameters hostRoomState and clientRoomState. (Look at https://github.com/wxyz-abcd/mooball/tree/main/examples/other/compareStates.js for the default implementation of desync checking.) This callback is called whenever a desync occurs if the host room's debugDesync value is also true. Defaults to null.

    OptionalidentityToken

    identityToken?: string

    A token that represents a user data in moo-hoo.com's database. If successful, room will create and emit an IdentityEvent for this player.

    Optionallibraries

    libraries?: Library[]

    An array of Library objects to be used. the objects should be derived from the provided Library class. Default value is []. Look at https://github.com/wxyz-abcd/mooball/tree/main/examples/libraries for example Library's to use here.

    OptionalnoPluginMechanism

    noPluginMechanism?: boolean

    If true, renderer and plugin mechanism will not work. Should only be used for optimal performance. You have to define Room._onXXXXXX callbacks by yourself.

    Optionalplugins

    plugins?: Plugin[]

    An array of Plugin objects to be used. the objects should be derived from the provided Plugin class. Default value is []. Look at https://github.com/wxyz-abcd/mooball/tree/main/examples/plugins for example Plugins to use here.

    OptionalproxyAgent

    proxyAgent?: ProxyAgentLike

    A custom proxy agent to use for the room's connection. This method does not work in browsers. Defaults to null.

    Optionalrenderer

    renderer?: Renderer

    The Renderer object that will render the game. The object should be derived from the provided Renderer class. Default value is null. Look at https://github.com/wxyz-abcd/mooball/tree/main/examples/renderers for example Renderers to use here.

    Optionalstorage

    storage?: Storage

    An object that stores information about the current player preferences.

    Optionalversion

    version?: int

    Mooball's version number. Other clients cannot join this room if their version number is different than this number. Default value is 9.
